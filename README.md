## 🎮 Управление

### Мобильные устройства

| Жест                          | Действие                  |
|-------------------------------|---------------------------|
| ⬅️➡️ Горизонтальный свайп     | Движение влево/вправо     |
| ⬆️ Вертикальный свайп вверх   | Поворот фигуры            |
| ⬇️⬇️ Резкий свайп вниз        | Hard drop (сброс)         |

### Клавиатура (ПК)

| Клавиша              | Действие                        |
|----------------------|---------------------------------|
| ← →                  | Движение                        |
| ↑                    | Поворот                         |
| ↓ (удержание)        | Ускоренное падение              |
| Пробел               | Hard drop (мгновенный сброс)    |
