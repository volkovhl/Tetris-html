# 🧩[TETRIS HTML](https://volkovhl.github.io/Tetris-html/)  
  

Классический Тетрис с уникальным управлением и динамической скоростью. Оптимизирован для мобильных устройств и ПК.

---

## ✨ Особенности

- 🎮 **Увеличенное поле** — 20×40 для более сложной и продолжительной игры
- 📱 **Сенсорное управление** — drag & drop с динамической чувствительностью поворота
- ⌨️ **Клавиатура** — стрелки + пробел (скорость перемещения синхронизирована с игровой)
- ⚡ **Регулировка скорости** — от 0.5x до 10x в реальном времени
- 🎯 **Умный lock delay** — льготный период перед фиксацией фигуры
- 🎨 **Современный дизайн** — темная тема, неоновые акценты
- 📱 **Адаптация** — поддержка safe areas, запрет зума

---

## 🎮 Управление

### Мобильные устройства

| Жест | Действие |
|------|----------|
| ⬅️➡️ Горизонтальный свайп | Движение влево/вправо |
| ⬆️ Вертикальный свайп вверх | Поворот фигуры |
| ⬇️⬇️ Резкий свайп вниз | Сброс фигуры |


### Клавиатура (ПК)

| Клавиша | Действие |
|---------|----------|
| ← → | Движение |
| ↑ | Поворот |
| ↓ (удержание) | Ускоренное падение |
| Пробел | Сброс фигуры |

---

## ⚙️ Технические детали

- Чистый HTML/CSS/JavaScript (без зависимостей)
- Canvas рендеринг
- Динамическая чувствительность поворота: `sensitivity = 150 / speedMultiplier`
- Адаптивная скорость перемещения при удержании клавиш
- Размер поля: 20x40 (в 2 раза выше классического)
